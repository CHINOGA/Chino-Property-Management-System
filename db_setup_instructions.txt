Rent Payment Reminder Script Scheduling Instructions:

1. Manual Execution:
   - You can manually run the rent payment reminder script using the helper script:
     php scripts/run_rent_payment_reminder.php

2. Scheduling on Windows (Task Scheduler):
   - Open Task Scheduler.
   - Create a new Basic Task.
   - Set the trigger (e.g., daily at a specific time).
   - Set the action to "Start a program".
   - Program/script: Path to your PHP executable (e.g., C:\php\php.exe).
   - Add arguments: c:/xampp/htdocs/Chino-Property-Management-System/scripts/rent_payment_reminder.php
   - Start in: c:/xampp/htdocs/Chino-Property-Management-System/scripts
   - Finish and save the task.

3. Scheduling on Linux (Cron):
   - Open crontab with: crontab -e
   - Add a line like:
     0 9 * * * /usr/bin/php /path/to/Chino-Property-Management-System/scripts/rent_payment_reminder.php
   - Save and exit.

4. Testing:
   - Run the helper script manually to verify it executes without errors.
   - Check the notifications table for new rent payment reminders.
   - Verify SMS messages are sent (check logs or NextSMS dashboard).
   - Verify notifications appear in the Notifications page in the UI.

5. Troubleshooting:
   - Ensure PHP CLI is installed and accessible.
   - Check file permissions for the script.
   - Review error logs for any issues.

Please let me know if you need assistance with any of these steps.
